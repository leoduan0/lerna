# Imperfect

The **imperfect** (imperfecto) tense is used for actions or descriptions which are incomplete.

It is often used for:

- Descriptions (_e.g._ weather, time, age, mood)
- Interrupted actions (_e.g._ while I was teaching the class, she interrupted me)

|                 | -ar     | -er/-ir | Ser     | Ir      |
| --------------- | ------- | ------- | ------- | ------- |
| **Yo**          | -aba    | ía      | -era    | -iba    |
| **Tú**          | -abas   | ías     | -eras   | -ibas   |
| **Él/Ella**     | -aba    | ía      | -era    | -iba    |
| **Nosotros**    | -ábamos | íamos   | -eramos | -íbamos |
| **Vosotros**    | -abais  | -íais   | -erais  | -ibais  |
| **Ellos/Ellas** | aban    | ían     | -eran   | -iban   |
