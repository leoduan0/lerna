# Present Indicative

The **present indicative** tense (el presente de indicativo) is used for current actions or descriptions.

|                 | -ar   | -er   | -ir   |
| --------------- | ----- | ----- | ----- |
| **Yo**          | -o    | -o    | -o    |
| **Tú**          | -as   | -es   | -es   |
| **Él/Ella**     | -a    | -e    | -e    |
| **Nosotros**    | -amos | -emos | -imos |
| **Vosotros**    | -áis  | -éis  | -ís   |
| **Ellos/Ellas** | -an   | -en   | -en   |
