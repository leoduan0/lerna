# Span

## Linear Combination

<Rule
  statement={
    <>
      <p>
        Given a set of vectors{' '}
        <Math math="S = \{v_1, v_2, v_3, ... , v_n\}" type="inline" />, the{' '}
        <strong>linear combination</strong> of <Math math="S" type="inline" />{' '}
        is:
      </p>
      <Math math="\sum_{i=1}^n \alpha_i v_i" type="block" />
    </>
  }
  terms={[
    {
      term: 'a_i',
      annotation: <span>Any arbitrary scalar</span>,
    },
    {
      term: 'v_i',
      annotation: (
        <span>
          The <Math math="i" type="inline" />
          -th vector in <Math math="S" type="inline" />
        </span>
      ),
    },
  ]}
  annotation={
    <p>
      Linear combination involves taking a set of vectors and scaling each
      vector in any arbitrary way using any arbitrary scalar.
    </p>
  }
/>

<Block text="Example">

Given $$ S = \{v_1, v_2\} $$, where $$ v_1 = \langle 1, 2, 3 \rangle $$ and $$ v_2 = \langle 4, 5, 6 \rangle $$, its linear combination is $$ a_1 \langle 1, 2, 3 \rangle + a_2 \langle 4, 5, 6 \rangle $$

</Block>

### Linear Dependence

<Rule
  statement={
    <>
      <p>
        Given a set of vectors{' '}
        <Math math="S = \{v_1, v_2, v_3, ... , v_n\}" type="inline" />, it is
        said to be <strong>linear dependent</strong> iff{' '}
        <Math math="\sum_{i=1}^{n} \alpha_i v_i = 0" type="inline" /> where at
        least one <Math math="\alpha_i \neq 0" type="inline" />
      </p>
    </>
  }
  terms={[
    {
      term: 'a_i',
      annotation: <span>Any arbitrary scalar</span>,
    },
    {
      term: 'v_i',
      annotation: (
        <span>
          The <Math math="i" type="inline" />
          -th vector in <Math math="S" type="inline" />
        </span>
      ),
    },
  ]}
  annotation={
    <p>
      A set of vectors is said to be <strong>linearly dependent</strong> if it
      contains components that, if removed, do not decrease the span of the set.
      In other words, if a set of vector is linearly dependent, then it contains
      at least one vector that is either a zero vector and/or a vector that is a
      linear combination of another/others. Otherwise, the set is said to be{' '}
      <strong>linearly independent</strong>.
    </p>
  }
/>

## Span

<Rule
  statement={
    <>
      <p>
        Given a set of vectors{' '}
        <Math math="S = \{v_1, v_2, v_3, ... , v_n\}" type="inline" />, the{' '}
        <strong>span</strong> of <Math math="S" type="inline" /> is:
      </p>
      <Math
        math="\operatorname{span}(S) = \left\{\sum_{i=1}^{n} \alpha_i v_i \middle| v_i \in S \right\}"
        type="block"
      />
    </>
  }
  terms={[
    {
      term: 'a_i',
      annotation: <span>Any arbitrary scalar</span>,
    },
    {
      term: 'v_i',
      annotation: (
        <span>
          The <Math math="i" type="inline" />
          -th vector in <Math math="S" type="inline" />
        </span>
      ),
    },
  ]}
  annotation={
    <>
      <p>
        Given a set of vectors, all possible linear combinations of that set is
        called the span of that set of vectors.
      </p>
      <p>
        In vector space, the span can also be thought as the amount of points
        which the given set of vectors can possibly reach already of by scaling
        it with some scalar.
      </p>
    </>
  }
/>
