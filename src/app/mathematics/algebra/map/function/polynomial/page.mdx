# Polynomial

An $$ n $$-th degree **polynomial** takes the form:

<Align type={'center'}>

$$ P(x) = a_n x^n + a_{n-1} x^{n-1} + a_{n-2} x^{n-2} + a_{n-3} x^{n-3} + ... + a_1 x + a_0 $$

</Align>

$$ a_n, a_{n-1}, ... $$ is often replaced with letters of the alphabet.

## Linear Function

A **linear function** takes the form:

<Align type={'center'}>$$ f(x) = ax + b $$</Align>

$$ x $$ can be found using:

<Align type={'center'}>$$ x = \frac{-b}{a} $$</Align>

## Quadratic Function

A **quadratic function** takes the form:

<Align type={'center'}>$$ f(x) = a x^2 + b x + c $$</Align>

$$ x $$ can be found using:

<Align type={'center'}>$$ x = \frac{-b \pm \sqrt{b^2-4ac}}{2a} $$</Align>

## Cubic Function

A **cubic function** takes the form:

<Align type={'center'}>

$$ f(x) = a x^3 + b x^2 + c x + d $$

</Align>

$$ x $$ can be found using:

<Align type={'center'}>
  $$ x = \sqrt[3]{\left(\frac{-b^3}{27a^3} + \frac{bc}{6a^2} - \frac{d}{2a} \right) + \sqrt{\left(\frac{-b^3}{27a^3} + \frac{bc}{6a^2} - \frac{d}{2a} \right)^2 + \left(\frac{c}{3a} - \frac{b^2}{9a^2}\right)^3}} + \sqrt[3]{\left(\frac{-b^3}{27a^3} + \frac{bc}{6a^2} - \frac{d}{2a} \right) - \sqrt{\left(\frac{-b^3}{27a^3} + \frac{bc}{6a^2} - \frac{d}{2a} \right)^2 + \left(\frac{c}{3a} - \frac{b^2}{9a^2}\right)^3}} - \frac{b}{3a} $$
</Align>
