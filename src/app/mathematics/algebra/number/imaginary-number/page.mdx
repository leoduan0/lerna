# Imaginary Number

**Imaginary numbers** arise from the solutions to the square root of negative numbers.

<Block text="Definition">

**Imaginary number**: $i=\sqrt{-1}$

</Block>

Using this definition, $i$ can be used to denote solutions to the square root of any negative numbers.
