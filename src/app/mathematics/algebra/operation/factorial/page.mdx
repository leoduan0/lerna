# Factorial

A **factorial** of a number $$ x $$, denoted by $$ x! $$, is given by the following:

<Rule
  statement={
    <Math
      math={'(x-1)! = \\Gamma(x) = \\int_{0}^{\\infty} t^{x-1} e^{-t} \\,dt'}
      type={'block'}
    />
  }
  annotation={
    <p>
      This is also known as the <strong>gamma function</strong>.
    </p>
  }
/>

When $$ x $$ is an integer:

<Rule statement={<Math math={'x!=\\prod_{i=1}^x n'} type={'block'} />} />
