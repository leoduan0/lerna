# Logarithm

<Block text="Definition">

$$ \log_{a}(b) = c \implies a^c = b $$

</Block>

## Identities

- $\log_{a}(b) = c \implies {a}^{c} = b$
- $\log_{e}(a) = \ln(a)$
- $\log_{10}(a) = \log(a)$
- $\log_{a}(c^d) = d \log_{a}(c)$
- $\log_{a}(bc) = \log_{a}(b) + \log_{a}(c)$
- $\log_{a}(b) = \frac{\log_{x}(b)}{\log_{x}(a)} , \forall x $
- $\log_{a}\left(\frac{b}{c}\right) = \log_{a}(b) - \log_{a}(c)$
- $a^{\log_{a}(b)} = b$
- $-\log_{a}(b) = \log_{b}(a)$
- $\log_{a}(1) = 0$
- $\log_{a}(a) = 1$
- $a^b = e^{b\ln(a)}$
