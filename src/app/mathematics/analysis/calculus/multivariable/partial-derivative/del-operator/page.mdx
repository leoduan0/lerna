## Gradient

Given a function $$ f $$, its $$ gradient $$ is:

<Rule
  statement={
    <Math
      math={
        '\\operatorname{grad}(f) = \\nabla f = \\vec{x}\\frac{\\partial f}{\\partial x} + \\vec{y}\\frac{\\partial f}{\\partial y} + \\vec{z}\\frac{\\partial f}{\\partial z}'
      }
      type="block"
    />
  }
  terms={[
    {
      term: '\\operatorname{grad}(f)',
      annotation: (
        <span>
          <strong>Gradient</strong> of <Math math="f" type="inline" />
        </span>
      ),
    },
  ]}
/>

## Divergence

Given a vector $$ \vec{v} = \langle v_1, v_2, v_3 \rangle $$, its $$ divergence $$ is:

<Rule
  statement={
    <Math
      math={

## Curl

Given a vector $$ \vec{v} = \langle v_1, v_2, v_3 \rangle $$, its **curl** is:

<Rule
  statement={
    <Math
      math={
        '\\operatorname{curl}(\\vec{v}) = \\nabla \\times \\vec{v} = \\det{\\begin{bmatrix} \\vec{x} & \\vec{y} & \\vec{z} \\\\ \\frac{\\partial}{\\partial x} & \\frac{\\partial}{\\partial y} & \\frac{\\partial}{\\partial z} \\\\ v_1 & v_2 & v_3 \\end{bmatrix}}'
      }
      type="block"
    />
  }
  terms={[
    {
      term: '\\operatorname{curl}(\\vec{v})',
      annotation: (
        <span>
          Curl of <Math math="\\vec{v}" type="inline" />
        </span>
      ),
    },
  ]}
/>

## Laplacian

Given a vector $$ \vec{v} = \langle v_1, v_2, v_3 \rangle $$, its **Laplacian** is:

<Rule
  statement={
    <Math
      math={
        '\\operatorname{div}(\\operatorname{grad}(\\vec{v})) = \\nabla \\cdot \\nabla \\vec{v} = \\nabla^2 \\vec{v} = \\vec{x}\\frac{\\partial^2 v_1}{{\\partial x}^2} + \\vec{y}\\frac{\\partial^2 v_2}{{\\partial y}^2} + \\vec{z}\\frac{\\partial^2 v_3}{{\\partial z}^2}'
      }
      type="block"
    />
  }
  terms={[
    {

      annotation: (

    },
  ]}
/>
