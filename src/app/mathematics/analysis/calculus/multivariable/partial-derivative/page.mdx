# Partial Derivative

**Partial derivatives** are very similar to derivatives in single variable calclulus, except that to find it, we will, for each variable of the function, allow it to vary while holding all other variables still by treating them as a constant.

For example, for $$ f(x, y) = 2x^3y^5 $$, its partial derivatives are as follows:

<Align type="center">
  <Math
    math={
      '\\frac{\\partial f(x, y)}{\\partial x} = {f}_{x}(x, y) = 6{x}^{2}{y}^{5}'
    }
    type={'block'}
  />
  <Math
    math={
      '\\frac{\\partial f(x, y)}{\\partial y} = {f}_{y}(x, y) = 10{x}^{3}{y}^{4}'
    }
    type={'block'}
  />
</Align>

In the former, we wanted to find the derivative of $$ f(x, y) $$ with respect to $$ x $$, and we did so by holding the other variable $$ y $$ still by treating it as a constant.

In the latter, we wanted to find the derivative of $$ f(x, y) $$ with respect to $$ y $$, and we did so by holding the other variable $$ x $$ still by treating it as a constant.

Like single variable derivatives, multivariable derivatives can also have higher orders:

<Align type="center">
  <Math
    math={
      '\\frac{\\partial}{\\partial x} \\left(\\frac{\\partial f(x, y)}{\\partial x}\\right) = \\frac{{\\partial}^2 f(x, y)}{{\\partial x}^2} = f_{xx}(x, y)'
    }
    type={'block'}
  />
  <Math
    math={
      '\\frac{\\partial}{\\partial y} \\left(\\frac{\\partial f(x, y)}{\\partial x}\\right) = \\frac{{\\partial}^2 f(x, y)}{\\partial y \\partial x} = {f}_{xy}(x, y)'
    }
    type={'block'}
  />
  <Math
    math={
      '\\frac{\\partial}{\\partial y} \\left(\\frac{\\partial f(x, y)}{\\partial y}\\right) = \\frac{{\\partial}^2 f(x, y)}{{\\partial y}^2} = f_{yy}(x, y)'
    }
    type={'block'}
  />
  <Math
    math={
      '\\frac{\\partial}{\\partial x} \\left(\\frac{\\partial f(x, y)}{\\partial y}\\right) = \\frac{{\\partial}^2 f(x, y)}{\\partial x \\partial y} = f_{yx}(x, y)'
    }
    type={'block'}
  />
</Align>

And so on.
