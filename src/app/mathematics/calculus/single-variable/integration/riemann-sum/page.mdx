# Riemann Sum

**Riemann sums** are types of approximations for the integral of a function. There are six types of Riemann Sums:

- Left
- Right
- Midpoint
- Upper
- Lower
- Trapezoidal

For each of the following sums, we divide $$ f(x) $$ into an $$ n $$ amount of $$ \Delta x $$ bounded by lines $$ x=x_n $$.

## Left

<Rule
  statement={
    <Math math="L(n) = \sum_{i=1}^{n} f(x_{i-1}) \Delta x_i" type="block" />
  }
/>

## Right

<Rule
  statement={
    <Math math="L(n) = \sum_{i=1}^{n} f(x_i) \Delta x_i" type="block" />
  }
/>

## Midpoint

<Rule
  statement={
    <Math
      math="L(n) = \sum_{i=1}^{n} f\left(\frac{x_{i-1}+x_i}{2}\right) \Delta x_i"
      type="block"
    />
  }
/>

## Upper

<Rule
  statement={
    <Math
      math="U(n) = \sum_{i=1}^n (\sup f([x_{i-1},x_i])) \Delta x_i"
      type="block"
    />
  }
/>

## Lower

<Rule
  statement={
    <Math
      math="L(n) = \sum_{i=1}^{n} (\inf f([x_{i-1},x_i])) \Delta x_i"
      type="block"
    />
  }
/>

## Trapezoidal

<Rule
  statement={
    <Math
      math="L(n) = \sum_{i=1}^{n} \frac{f(x_{i-1})+f(x_i)}{2} \Delta x_i"
      type="block"
    />
  }
/>
