# Area

**Area** is the amount of space a 2-dimensional shape takes.

## Polygon

<Rule
  statement={
    <Math
      math={
        'A_{polygon} = \\frac{n}{4}\\cot\\left(\\frac{180\\degree}{n}\\right)l^2'
      }
      type="block"
    />
  }
  terms={[
    {
      term: 'A_{polygon}',
      annotation: (
        <span>
          Area of an <Math math="n" type="inline" />
          -sided polygon
        </span>
      ),
    },
    {
      term: 'n',
      annotation: <span>Number of sides</span>,
    },
    {
      term: 'l',
      annotation: <span>Length of a side</span>,
    },
  ]}
/>

This includes triangles and squares, but since the coefficient is 1 for a square:

<Rule
  statement={<Math math="A_{square} = l^2" type="block" />}
  terms={[
    {
      term: 'A_{square}',
      annotation: <span>Area of a square</span>,
    },
    {
      term: 'l',
      annotation: <span>Length of a side</span>,
    },
  ]}
/>
