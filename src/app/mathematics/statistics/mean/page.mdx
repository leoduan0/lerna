# Mean

## Arithmetic Mean

<Rule
  statement={
    <Math
      math="\\overline{x}_{arithmetic} = \\frac{\\sum_{i=1}^n x_i}{n}"
      type="block"
    />
  }
  terms={[
    {
      term: '\\overline{x}_{arithmetic}',
      annotation: <span>Arithmetic mean</span>,
    },
    {
      term: 'x_i',
      annotation: (
        <span>
          <Math math="i" type="inline" />
          -th value of population
        </span>
      ),
    },
    {
      term: 'n',
      annotation: <span>Population size</span>,
    },
  ]}
  annotation={
    <span>
      This is also known as the <strong>average</strong>.
    </span>
  }
/>
